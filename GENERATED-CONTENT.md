# Generated Content — OSS Prototype 2

Реестр всего, что добавлено в прототип сверх того, что есть в `docs/oss-prototype/spec/` и `docs/oss-prototype/design/refs/`. Это нужно для проверки и итеративной доработки: Юрий просматривает каждую запись, что-то правит, что-то помечает как disabled.

## Принципы генерации

- Тексты — по редполитике Госуслуг (`research/gosuslugi-editorial-policy-official.md`) и Ильяхов (короткие, конкретные, без воды).
- UX-элементы — в рамках компонентной базы из `shared/components.md` и `shared/patterns.md`.
- Domain-факты — сверяться с `knowledge/oss-prototype/` (legal-frame, audit-findings) и при необходимости гуглить.

## Структура записи

```
### [Короткий ярлык]
- **Локация:** модуль / экран / компонент
- **Тип:** текст / UI-элемент / демо-данные
- **Почему:** для чего добавил
- **Содержимое:** сам текст или описание
- **Статус:** новое / проверено / disabled
```
